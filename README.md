# DeepBrux: Wearable Bruxism Detection System

## Overview
A wearable embedded system for detecting bruxism-related jaw movements using a piezoelectric sensor and machine learning.

## Hardware
- Teensy 4.1
- Piezoelectric sensor

## Features
- Built wearable device 
- Real-time data acquisition (200 Hz)
- Signal preprocessing (filtering, segmentation)
- Spectrogram-based feature extraction
- ResNet-18 classification model (86% accuracy)

## Pipeline
1. Hardware Assembly
2. Data acquisition
3. Preprocessing
4. Feature extraction
5. Classification

## Results
Achieved 86% accuracy across 5 classes.
